## Google Search Console 登録手順

1. https://search.google.com/search-console/ にアクセス
2. Googleアカウントでログイン
3. 「プロパティを追加」→「URLプレフィックス」を選択
4. `https://aiworkstage.com/` を入力
5. 確認方法を選択（HTMLファイル or DNSレコード推奨）
6. 確認完了後、「サイトマップ」メニューから `https://aiworkstage.com/sitemap.xml` を送信

**必要なもの：** aiworkstage.com ドメインに紐づいたGoogleアカウント、またはDNSレコード編集権限
